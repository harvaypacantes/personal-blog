HTML & CSS for First-Year BSIT Students - A Beginner-Friendly Guide

Harvay Alfred | First-year BSIT

Why learn HTML and CSS?

HTML and CSS are the basic tools for creating websites. HTML gives the structure (headings, paragraphs, images, links), while CSS controls how the page looks (colors, spacing, layout). As a first-year BSIT student, mastering these two languages helps you build projects, create a portfolio, and understand how the web works.

HTML basics - the structure

HTML uses tags surrounded by angle brackets. Most tags come in pairs: an opening tag and a closing tag. Here is a tiny example of a simple webpage structure: 

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello, BSIT!</h1>
    <p>This is my first web page.</p>
  </body>
</html>

Key tags to learn first: <h1>-<h6> (headings), <p> (paragraph), <a> (link), <img> (image), and <ul>/<ol> (lists).

CSS basics - the style

CSS uses selectors to target HTML elements and apply styles. Here's how to change text color and center a heading: 


          h1 {
            color: blue;      /* changes the text color*/
            text-align:center;     /*centers the heading*/
          }

You can add CSS inside the HTML using a <style> tag, or place it in a separate file (recommended). Learn simple layout tools first: flexbox and grid.

SEPERATE CSS FILE


           h1 {
            color: blue;      
            text-align:center;    
          }
          .flex-container{
            display:flex;
            justify:content;
            gap:20px;
          }
          .grid-container{
            display:grid;
            gap:5px;
          }

ADDING CSS INSIDE HTML


          <link rel="stylesheet/sample" href="sample/style.css>
          <h1>Learning layouts in HTML and CSS>h1>

A small example - HTML + CSS together

Copy this into a file named index.html and open it in your browser:

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width,initial-scale=1">
  <title>My First Page</title>
  <style>
    body{font-family:Arial, sans-serif;padding:20px}
    .card{border:1px solid #ddd;padding:16px;border-radius:8px;max-width:600px}
    .card h2{color:#2563eb}
  </style>
</head>
<body>
  <div class="card">
    <h2>My First Card</h2>
    <p>This page shows HTML and CSS in one file.</p>
  </div>
</body>
</html>
Tip: Save the file, then double-click it to open in your browser. Try changing text and styles and refresh the page to see changes.

A simple learning path for BSIT students

  1. Understand HTML tags and semantic elements (<nav>, <main>, <header>, <footer>).
  2. Learn CSS basics: selectors, box model, margin/padding, fonts, colors.
  3. Practice layout: flexbox first, then grid.
  4. Build small projects: a personal homepage, a nav bar, and a simple portfolio.
  5. Share your projects-upload them to GitHub Pages.

Final thoughts

HTML and CSS are friendly and rewarding to learn. Start small, practice often, and don't be afraid to break things-debugging is how you learn. As a BSIT student, these skills will make your school projects and future job applications stand out.
