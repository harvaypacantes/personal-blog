How we set up a beginner-friendly dev environment

By Harvay & Alfred

Why a dev environment matters

When you're learning to code, a simple and predictable environment saves time and confusion. Below we list tools we use and why.

1. Code editor

We use VS Code - lightweight, extensible, and beginner-friendly. Install useful extensions like Live Server, Prettier, and an HTML/CSS helper extension.

2. Version control (basic)

Learn git for saving snapshots of your work. Start with commits, branches, and pushing to GitHub.

Tip: Create a README for each small project. It helps you remember what you built and why.

3. Practice projects

Build tiny projects: a to-do app, a small portfolio, an attendance script. Each project teaches a different concept.

Final notes

Remember: consistency > intensity. Build a little each week and document what you learn.
